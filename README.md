## Features

## Job App Backend

- **Company Management**:
  - **Add** a new company
  - **Delete** an existing company
  - **Update** company details

- **Job Management**:
  - **Add** a new job
  - **Delete** an existing job
  - **Update** job details

- **Review Management**:
  - **Add** a review for a company
  - **Delete** an existing review
  - **Update** review details
